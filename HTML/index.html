Introducción a CSS

CSS es como se visualiza el contenido HTML (lo bonito). Le da la presentación al documento HTML. No confundir con lenguajes de programación, no lo son y aunque no sea un lenguaje de programación, hay ciertas caracteristicas y/o funciones que permiten aplicar ciertos comportamientos y, hasta ciertos punto, automatizar y/o utilizar algo parecido a la programación. Se pueden hacer animaciones justo también con CSS.

Se debe juntar el CSS junto al HTML y la programación JS.

Sintaxis basica

Una regla CSS consta de dos partes:

  1. El selector: Es el elemento HTML al que nosotros podemos aplicarle cierto bloque de selección de estilos (cualquier etiqueta).
  2. Bloque de declaraciones: Cada uno de los atributos que vamos a modificar. Cada atributo que modificamos tendra la siguiente sintaxis -atributo: valor; (esto sera delimitado por los "{}"). 
    
                Ejemplo:
             
                h1 {
                    color: orangered;
                    background-color: skyblue;
                }

Selectores basicos

Existen 3 selectores basicos

  1. Etiquetas - Son elementos HTML
  2. Identificadores - Accedemos a traves del atributo HTML id (#)
  3. Clases - Atributo class (.)

Selector Universal (*)

El selector universal, representado por un asterisco (*), aplica estilos a todos los elementos del documento HTML, a menos de que se este especificando una clase, en dado caso, solo se aplicaria el estilo a las etiquetas seleccionadas por la clase

Pseudoclases

Las pseudoclases en CSS son selectores especiales que te permiten aplicar estilos a elementos HTML en función de su estado o comportamiento. A diferencia de las clases y los IDs, que se basan en atributos HTML, las pseudoclases se basan en el estado dinámico de un elemento, como por ejemplo:

  • Si el cursor del mouse está sobre el elemento: :hover
  • Si el elemento está enfocado: :focus
  • Si el elemento es un enlace: :link, :visited
  • Si el elemento es el primero de su tipo: :first-child
  • Si el elemento es el último de su tipo: :last-child
  • Si el elemento es un hijo par: :nth-child(even)
  • Si el elemento es un hijo impar: :nth-child(odd)
  • Si el elemento está vacío: :empty
  • Si el elemento está seleccionado: :checked
  • Si el elemento está deshabilitado: :disabled

Funcionamiento del algoritmo de CSS

El algoritmo de CSS: Es la forma en que el navegador aplica los estilos al documento HTML. Es vital entender este concepto para que se entienda el como se aplica y en algunas ocasiones se sobreescriben las reglas CSS.

  1. La cascada:

    Es el mecanismo en que el navegador aplica los estilos. Toma en cuenta lo siguiente en este orden:

    • El user Agent (los estilos por defecto del navegador)
    • Personalizaciones del usuario (del usuario haciendo uso del navegador)
    • Los estilos de autor

  2. La especificidad:

    La especificidad es el peso que tiene un selector cando hay conflicto de estilos. Se calcula de la siguiente forma:

    • Etiquetas y pseudoelementos -------- 0,0,0,1
    • Clases, atributos y pseudoclases --- 0,0,1,0
    • Identificadores -------------------- 0,1,0,0
    • Estilos en línea ------------------- 1,0,0,0
    • !important ------------------------- rompe la especificidad

  3. La herencia

    La herencia, es la capacidad de un selector de obtener (heredar) los valores de sus ancestros más cecanos, para aplicarla se usa el valor inherit, si queremos evitarla podemos asignar otro valor o inicializar la propiedad en cuestión con el valor initial.

    4. Más vale estar solo que mal acompañado. - Anon

Modelo de caja

Todo en CSS tiene una caja alrededor. Es la forma en que CSS interpreta las etiquetas HTML. Para CSS todo elemento de HTML son una caja (caja de linea o caja de bloque). 

1. El contenido (content): El contenido (texto) del elemento HTML
2. El borde (border): Lo que delimita cada uno de los elementos HTML
3. El relleno (padding): Son las distancias internas (La distancia del borde al contenido)
4. El márgen (margin): Son las distancias externas (La distancia entre el elemento html y sus elementos hermanos o padres)
 Dentro del modelo de caja hay que considerar que una caja tendrá 4 lados:
        
            1. Arriba (top)
            2. Derecha (right)
            3. Abajo (bottom)
            4. Izquierda (left)

Posicionamiento en CSS

Las propiedades FLOAT y POSITION nos permiten modificar la posicion natural de cualquier elemento del documento HTML.

El navegador coloca cada elemento teniendo en cuenta el orden en el que aparece en el documento y su tipo de visualización dependiendo si es un elemento de línea o de bloque.

Con FLOAT y POSITION podemos modificar este comportamiento.

Elementos flotantes

  • Float: Convierte un elemento en flotante desplazandolo hasta la zona más a la izquierda o más a la derecha de la posición en la que originalmente se encontraba.
  • Clear: limpia la flotación (left, right, both)

Tipos de Posicionamiento:

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky

Para mover los elementos posicionades se activan las propiedades:

  • top (vertical - eje y)
  • bottom (vertical - eje y)
  • left (horizontal - eje x)
  • right (horizontal - eje x)
  • z-index (profundidad - eje z)

Las propiedades top y left van a tener preferencia por sobre bottom y right respectivamente

Estas 5 propiedades no funcionan con el valor de static
