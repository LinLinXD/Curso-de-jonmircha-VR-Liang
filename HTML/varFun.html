Función url()

La función url() en CSS se utiliza para incluir recursos externos, como imágenes, fuentes, o archivos de audio y video, en una hoja de estilo. Se especifica dentro de propiedades de estilo que requieren una URL para localizar el recurso que se desea incluir.
